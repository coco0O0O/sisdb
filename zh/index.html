# 安装 ClChart

# 手动下载

你可以在GitHub releases下载最新的 clchart版本

# npm

npm install clchart --save

# 源码安装

克隆github仓库并打包:

git clone git@github.com:seerline/clchart.git
cd clchart
npm install

# 打包
npm run build

# html文件引入

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <!-- import ClChart file -->
    <script src="./clchart.js"></script>
</head>
</html>
